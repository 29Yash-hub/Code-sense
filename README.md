# CodeSense - Full Professional Animated (client+server)

This archive contains a production-ready starter for CodeSense (React + Vite frontend, Node/Express backend).

## Quick start

1. Server:
   cd server
   npm install
   cp .env.example .env
   # add your OPENAI_API_KEY to .env file
   npm start

2. Client:
   cd client
   npm install
   npm run dev

Open http://localhost:5173 in your browser.
